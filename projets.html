Projet 103-203 : Installation d’une machine virtuelle

Pour ce projet nous devions apprendre à installer une machine virtuelle et comprendre les différents paramètres de configuration afin de mieux comprendre son fonctionnement. Pour ce faire nous devions installer un système Debian 64 bits et le configurer (manuellement et automatiquement), gérer les droits des utilisateurs et installer de nouveaux logiciels tels que git (Utilisation d'apt pour charger des logiciels depuis le miroir). De plus ce projet nous a introduit l'outil markdown afin de pouvoir faire des rapports structurés.

Compétences :

  • Lignes de Commandes Linux
  • Gestion des droits utilisateurs
  • Utilisation de gestionnaire de paquet
  • Redaction de Rapports Structurés via l'outils markdown
