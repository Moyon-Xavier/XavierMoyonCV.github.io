Projet 103-203 : Installation d’une machine virtuelle

Pour ce projet nous devions apprendre à installer une machine virtuelle et comprendre les différents paramètres de configuration afin de mieux comprendre son fonctionnement. Pour ce faire nous devions installer un système Debian 64 bits et le configurer (manuellement et automatiquement), gérer les droits des utilisateurs et installer de nouveaux logiciels tel que git (Utilisation de apt pour charger des logiciels depuis le miroir) De plus ce projet nous a introduit l'outil markdown afin de pouvoir faire des rapports structurés.

Compétences :

  • Lignes de Commandes Linux
  • Gestion des droits utilisateurs
  • Utilisation de gestionnaire de paquet
  • Redaction de Rapports Structurés via l'outils markdown

Points Forts :

  • Patience, il est compliqué de trouver la solution adéquate en linux lorsque l’on débute il faut savoir chercher
  • Créativité, afin de réaliser une vidéo originale explicative sur l’installation d’une machine virtuelle.
  • Avoir quelques connaissances en anglais car il existe des manuels linux mais la plupart sont en anglais
  • Savoir écrire, filmer et monter une vidéo

Projet 2.04 - exploitation d'une BDD

Pour ce projet nous devions importer les données des JO, et créer un modèle conceptuel de données (permettant d’organiser les données afin de faciliter les futurs requêtes mais aussi d’éviter les redondances), ventiler les données afin de les répartir dans les différentes tables, et effectuer des requêtes afin d’effectuer des analyses statistiques et une analyse en anglais sur des athlètes ayant des caractéristique extreme.   • Réaliser un compte de résultat

Points Forts :

  • Patience lors de la réalisation du rendu statistique (Il s'agit d'une étape longue et redondante)
